Tencent Releases Hy3: 295B-Parameter Open-Source MoE LLM

News time: 2026-07-07T02:27:41.106773+00:00 · Source: rss

Tencent has officially launched Hy3, a groundbreaking open-source large language model that pushes the boundaries of scale, efficiency, and accessibility in the LLM ecosystem. Built by Tencent’s Hy Team, Hy3 is a Mixture-of-Experts (MoE) architecture with a staggering total of 295 billion parameters — yet only 21 billion are activated per inference step. This selective activation enables high performance without the full computational burden of dense models of comparable size. The model also features a 256K-token context window, supporting exceptionally long inputs — ideal for document analysis, code generation, and complex reasoning tasks.
