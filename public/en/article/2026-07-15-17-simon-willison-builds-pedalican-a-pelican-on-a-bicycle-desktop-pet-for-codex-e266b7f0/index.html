Simon Willison builds 'Pedalican', a pelican-on-a-bicycle desktop pet for Codex

News time: 2026-07-15T04:15:47.545146+00:00 · Source: rss

In a delightful blend of developer whimsy and AI-powered tooling, Simon Willison has introduced Pedalican — a custom animated desktop pet for OpenAI’s Codex Desktop. The pet features a charming baby pelican riding a tiny sky-blue bicycle, bouncing across the screen while delivering real-time updates on Codex agent tasks. Willison discovered Codex’s newly launched ‘pet’ API almost by accident after stumbling upon a built-in robot pet — a nostalgic nod to Clippy — and quickly realized developers could extend it with their own creations.

What makes Pedalican especially notable isn’t just its playful design, but how it was made: entirely guided by natural language prompts fed to GPT-5.6 Sol (a version of OpenAI’s latest multimodal model), using the gpt-image-2 API to generate sprite assets. Willison meticulously documented every step — from the initial character reference image (a centered pelican-on-bike against magenta chroma-key background) to individual animation frames like waving, pedaling, and idle bouncing. Each GIF loop was compiled from single-frame PNGs generated via targeted prompts, then assembled into sprite sheets compatible with Codex’s pet runtime.

The project is fully open source and grounded in two Apache 2.0–licensed components from OpenAI’s official repositories: hatch-pet, a skill that scaffolds and validates pet definitions, and imagegen, a utility for generating and preprocessing pet assets. Willison also published exhaustive notes, prompt histories, intermediate images, and even debugging logs — turning Pedalican into both a functional pet and a transparent case study in AI-assisted asset creation for constrained UI contexts.

Beyond novelty, Pedalican highlights an emerging pattern: using generative AI not just for code or text, but for rapid, iterative visual prototyping in developer tools. Its success hinges on precise prompt engineering — specifying dimensions (192×208 px), chroma-key color (#FF00FF), pose clarity, and animation readiness — all requirements dictated by Codex’s pet specification. As community galleries like Petdex and codexpets.app begin to populate with user-submitted pets, projects like Pedalican set early standards for quality, documentation, and reproducibility in this new category of interactive developer companions.
