Simon Willison launches LLM cliché highlighter to detect AI writing patterns

News time: 2026-07-18T04:12:27.735507+00:00 · Source: rss

Frustrated by the repetitive, formulaic language increasingly common in AI-generated articles—phrases like 'no fluff, no filler, no jargon'—developer Simon Willison built a lightweight but incisive web tool: the LLM Cliché Highlighter. The tool scans text for eleven empirically observed stylistic patterns typical of large language model output, visually flagging them with color-coded highlights to help readers quickly spot telltale signs of machine authorship.

Unlike statistical or watermark-based detection methods, this tool takes a linguistic, human-centered approach. It doesn’t claim to definitively prove AI origin; instead, it surfaces recurring rhetorical tics—such as 'is real and…', 'worth naming', or 'not just X, but Y' constructions—that editors, educators, and critical readers have repeatedly noted across thousands of AI-written pieces. The interface is deliberately minimal: users paste text or input a URL (fetched via r.jina.ai’s reader API), then toggle options like 'Show just the highlights' to focus on flagged segments.

The tool’s design reflects Willison’s pragmatic philosophy: rather than chasing elusive technical 'truth detection', it empowers users with pattern literacy. Each match links to an explanation—for example, highlighting "'s worth naming" under the 'Worth naming' pattern, which signals a tendency toward performative solemnity common in LLM outputs. A sample analysis shows how even a short sentence like 'That loss is real and it's worth naming' triggers multiple overlapping flags—demonstrating how these clichés often cluster, amplifying their perceptual weight.

Importantly, the tool was developed using 'vibe coding' with Fable 5, Anthropic’s newly released high-performance coding model praised for its ability to generate production-ready code from natural-language prompts. This meta-layer—using cutting-edge AI to build a tool that critiques AI writing—adds irony and depth to the project. While not a forensic detector, the LLM Cliché Highlighter serves as a timely, pedagogical intervention in an era where distinguishing human voice from algorithmic mimicry is becoming essential for media literacy, editorial integrity, and ethical communication.
