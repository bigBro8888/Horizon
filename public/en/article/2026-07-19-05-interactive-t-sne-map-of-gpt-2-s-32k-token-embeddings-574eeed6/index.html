Interactive t-SNE Map of GPT-2’s 32k Token Embeddings

News time: 2026-07-19T04:32:10.864556+00:00 · Source: reddit

A new open tool offers an intuitive, mobile-friendly window into the static semantic architecture of GPT-2-small—without requiring inference, context, or model execution. Built around the model’s 32,070 alphabetic token embeddings from the Word Token Embedding (WTE) table, the visualization uses t-distributed stochastic neighbor embedding (t-SNE) to project high-dimensional vectors into an interpretable 2D layout. Unlike dynamic attention-based analyses, this map captures only the pre-trained geometric relationships among tokens—revealing how the model implicitly groups words like 'cat', 'dog', and 'lion' based on their raw embedding proximity, long before any text generation occurs.

The interface is deliberately tactile: users can pinch-to-zoom on touchscreens, tap any token to highlight its nearest neighbors, and then tap those neighbors to traverse the graph step-by-step—effectively walking a semantic path through the embedding space. Edges are not arbitrary; they form a minimum spanning tree (MST), meaning every visible connection represents a true nearest-kin relationship in the compressed t-SNE space, with no cycles and minimal total distance. A search bar further enables rapid navigation—type 'quantum' or 'Paris' to jump directly to relevant regions.

This isn’t just a pretty picture—it’s a pedagogical instrument. For students and researchers, it transforms abstract vector arithmetic into tangible spatial intuition: you can see why 'king' sits near 'queen' and 'monarch', why punctuation tokens cluster separately, and how rare or misspelled tokens drift toward the periphery. Because it runs entirely client-side and requires no backend or API calls, it works offline and preserves privacy—no tokens are sent anywhere. The underlying embedding table is frozen; no fine-tuning, no gradients, no forward pass—just pure geometry.

While large language models are often treated as black boxes, tools like this peel back one foundational layer: the static vocabulary-level semantics baked into initialization and pre-training. It complements—but does not replace—dynamic interpretability methods like attention visualization or feature attribution. As interest grows in model transparency and teaching AI fundamentals, such lightweight, interactive, and accessible visualizations fill a critical gap between theory and intuition.
