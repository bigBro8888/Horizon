Interactive Gaussian Splatting tour of Grace Cathedral launches in browser

News time: 2026-07-21T04:28:45.283836+00:00 · Source: hackernews

A photorealistic, interactive 3D reconstruction of Grace Cathedral in San Francisco is now accessible directly in modern web browsers — no downloads or plugins required. Built using Gaussian Splatting, a cutting-edge technique for real-time radiance field rendering, the model lets users freely navigate the cathedral’s soaring nave, stained-glass windows, and intricate stonework with smooth, high-fidelity visuals. The experience runs entirely client-side via WebGPU, leveraging the GPU for fast rasterization of thousands of anisotropic 3D Gaussians — each representing a tiny, oriented ellipsoid that emits light and color based on photographic input.

This project follows creator Vincent Woo’s earlier work scanning Sutro Tower with drone-captured imagery, demonstrating how Gaussian Splatting bridges professional photogrammetry and accessible cultural documentation. Unlike traditional mesh-based 3D models or slow neural radiance fields (NeRFs), Gaussian Splatting achieves real-time performance at 1080p resolution by optimizing compact, differentiable primitives — making it uniquely suited for web deployment where bandwidth, memory, and compute are constrained.

The implementation pushes current web standards: it relies on WebGPU — the successor to WebGL — which provides low-level, cross-platform GPU access through Vulkan, Metal, or Direct3D backends. While Chrome and Edge have supported WebGPU since April 2023, Firefox only added stable support in version 141 (July 2025), and Safari followed in June 2025. This timing explains why some users report instability: one Firefox user encountered buffer size limits (max_buffer_binding_size = 134,217,728 bytes) when loading the large splat dataset, causing crashes after a few seconds. In contrast, mobile users praised seamless performance — likely due to tighter memory management and efficient tile-based rendering on iOS and Android WebGPU implementations.

Beyond technical novelty, the project underscores a growing trend: using emerging graphics techniques to preserve and democratize access to architectural heritage. From Notre Dame’s VRML-era reconstructions to today’s Gaussian Splatting tours, the goal remains consistent — to make historically significant spaces explorable, educational, and emotionally resonant. As hardware and APIs mature, such experiences may soon become standard tools for museums, educators, and preservationists — all running inside the browser.
