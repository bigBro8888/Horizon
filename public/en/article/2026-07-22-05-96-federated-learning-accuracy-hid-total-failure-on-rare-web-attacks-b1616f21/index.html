96% federated learning accuracy hid total failure on rare web attacks

News time: 2026-07-22T04:31:11.986935+00:00 · Source: reddit

A researcher’s federated learning experiment on network intrusion detection revealed a sobering gap between headline metrics and real-world performance. Using the CICIDS2017 dataset—split across four simulated client silos by attack type—the project tested FedAvg, FedProx, and FedNova against a centralized baseline. Three silos contained abundant traffic data, but one—‘Web Attacks’—held just ~3,000 samples out of 3 million total, making it severely underrepresented. Despite this imbalance, FedAvg achieved an impressive-sounding 96% global accuracy. Yet when performance was broken down per silo, the model scored only 49% accuracy on Web Attacks—and critically, 0.00 recall: it missed every single instance of that attack category.

This isn’t just a statistical quirk—it’s a systemic blind spot. Global accuracy averages performance across all samples, so dominant classes drown out minority failures. In security contexts, missing all instances of a real-world attack type is catastrophic—even if overall accuracy looks stellar. The researcher further discovered that even the centralized ‘gold standard’ model showed alarming instability: across 10 random seeds, its recall on Web Attacks swung from 57% to 99.5%, proving that randomness alone can determine whether a rare class is detected at all.

FedNova, which normalizes local updates by step count rather than raw parameter averaging, delivered markedly more consistent results—maintaining high-90s accuracy and strong recall across all silos, including Web Attacks, without sacrificing global performance. This suggests aggregation mechanics—not just model architecture or data volume—are decisive for fairness and reliability in federated settings.

The takeaway is both technical and cultural: evaluating federated models solely on global accuracy invites dangerous overconfidence, especially in high-stakes domains like cybersecurity. Reporting must include per-client and per-class metrics—particularly recall, precision, and F1—for minority categories. As the author notes, this isn’t about discarding accuracy; it’s about refusing to let it obscure what matters most: whether the system actually protects against all threats, not just the most common ones.
