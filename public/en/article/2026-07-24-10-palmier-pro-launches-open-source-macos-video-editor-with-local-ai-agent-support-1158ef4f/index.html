Palmier Pro launches open-source macOS video editor with local AI agent support

News time: 2026-07-24T04:29:05.208269+00:00 · Source: hackernews

Palmier Pro is a new open-source video editor for macOS that embeds AI deeply into the editing workflow—not as a plugin or cloud add-on, but as an integrated, locally orchestrated assistant. Built by Marcos and Harrison, cofounders of Palmier, it emerged from their own frustration editing AI-generated launch videos: constantly shuttling files between AI tools and traditional editors broke creative flow. So they built a minimal, Swift-native editor where Claude or Codex can generate transitions, cut long-form content into shorts, and edit multicam timelines—all without leaving the app.

What sets Palmier Pro apart is its dual AI interaction model. Users can either chat directly inside the editor (low-latency, UI-adjacent) or connect external AI agents—like custom Claude or Codex setups—via a local MCP (Model Context Protocol) server. This isn’t just API access: the MCP server exposes full project control—importing media, searching footage using locally run SigLIP2 embeddings, editing tracks and keyframes, generating sound effects or captions, and exporting final videos. Real-world use cases already include automating podcast-style edits across large footage libraries and rapid iteration on AI-assisted rough cuts.

The editor runs entirely on-device for core operations: speech transcription via Apple’s SpeechAnalyzer, beat detection with beat_this, silence detection via Silero VAD, and frame embedding with SigLIP2—all offline and privacy-preserving. While AI generation features (e.g., AI transitions or clips) route through Palmier’s backend (requiring sign-up and free credits), the editor itself needs no login. It currently supports only macOS Sequoia (15.0+, labeled 'macOS 26' in the post), prioritizing performance and native integration over cross-platform compatibility.

Palmier Pro reflects a broader shift: AI isn’t replacing editors—it’s handling mechanical tasks like syncing multicam audio, finding B-roll by semantic search, or splitting hours of footage into TikTok-ready segments. As the team puts it, ‘AI is not very good at creative editing—but given a pattern, it does a decent job at rough cut.’ Their goal isn’t automation for its own sake, but lowering the barrier so more people can focus on storytelling, not timeline management.
