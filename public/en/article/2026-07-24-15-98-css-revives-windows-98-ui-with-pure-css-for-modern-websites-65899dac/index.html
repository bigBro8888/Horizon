98.css revives Windows 98 UI with pure CSS for modern websites

News time: 2026-07-24T04:29:05.208135+00:00 · Source: hackernews

A lightweight, open-source CSS framework called 98.css is bringing the chunky buttons, gray gradients, and animated taskbar of Windows 98 back to contemporary web development — not as a joke, but as a functional, accessible, and surprisingly thoughtful design alternative. Built entirely in CSS (no JavaScript required), it offers ready-to-use components like windows, title bars, checkboxes, radio buttons, scrollbars, and even a working status bar — all styled to match Microsoft’s iconic 1998 operating system down to pixel-perfect shadows and hover effects.

The project gained traction in 2020, resurfaced strongly in 2022 and again in late 2024 — each time sparking lively discussion on Hacker News and GitHub. Its creator, Jordan Scales, describes it as a ‘burnout recovery project’ born from nostalgia and frustration with modern design homogeneity. Rather than parodying retro aesthetics, 98.css treats them seriously: its components follow semantic HTML practices, support keyboard navigation, and avoid hardcoded colors — instead using CSS variables for easy theming and accessibility adjustments.

What makes 98.css stand out isn’t just visual fidelity — it’s how it invites reflection on interface evolution. Users point to practical strengths long abandoned: multi-row browser tabs that scale naturally, tactile feedback from depressed buttons, and windowed layouts that encourage spatial memory. As one commenter noted, ‘the way multi-row tabs Just Work in the UIs of yesteryear chef’s kiss’ — highlighting usability features lost in today’s flat, minimalist paradigms.

Beyond nostalgia, 98.css fits into a broader resurgence of retro-inspired tooling — from XFCE themes mimicking Windows XP to Neobrutalist React libraries like RetroUI. It doesn’t aim to replace modern frameworks but offers developers a deliberate stylistic counterpoint: a reminder that interface design choices carry historical weight, technical trade-offs, and emotional resonance. Whether used for a personal portfolio, an art project, or simply to poke fun at design dogma, 98.css proves that ‘old’ doesn’t mean obsolete — it means differently optimized.
